⭐ PROMPT SET FOR BUILDING THE “LEGACY BIO APP” MVP
PROMPT 1 — Project Setup & Architecture

Create a full-stack web application called Legacy BIO App for tracking biomarker assay samples from intake → processing → result approval.

Requirements:

Backend: Node.js + Express or Python + FastAPI (choose one and scaffold cleanly)

Frontend: React (desktop-first responsive)

Database: PostgreSQL (preferred) or MySQL

Auth: Secure username/password login + role-based access control
Roles: Technician, Manager, Admin

Use environment-based config and HTTPS-ready structure

Structure the repo for clarity:

/server

/client

Common README with install/run instructions

Include:

Initial user auth endpoints

Basic routing placeholders for all MVP screens:
/login, /intake, /worklist, /samples/:id, /dashboard

Focus on correctness, security, and auditability over visual polish.
