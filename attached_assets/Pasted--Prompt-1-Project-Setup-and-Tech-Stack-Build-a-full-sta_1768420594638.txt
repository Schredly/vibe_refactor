✅ Prompt 1 — Project Setup and Tech Stack

Build a full-stack web app called Legacy BIO that tracks biomarker assay samples from intake to approval.
Tech stack:

Backend: Node.js with Express

Database: PostgreSQL

Frontend: React with Material UI

Structure: Monorepo with /server and /client

No mobile support; web-only desktop app

No external integrations yet (LIMS stub only)

Initial project setup should include:

Clean folder structure

.env example for database config

Basic README

Login route, status board route, dashboard route placeholders

This is a regulated-style MVP: prioritize correctness, auditability, and clarity over UI flash.
